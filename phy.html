What is

「簡単に。清潔に。薬が塗れる」PHY

クラウドファンディングを行うなら？という仮定の元での製品提案


制作：グループワーク（5人）　　　　　　担当：ウェブデザイン / コーディング

参加時期：2021年11月～2021年12月　　　使用アプリ・技術：Figma / HTML,CSS / jQuery / VSCode / GitHub

Process

01. クラウドファンディングサイトでの分析

"Makuake"や"Campfire"などの国内のクラウドファンディングサイトで、成功している製品と失敗している製品を分析し、何が要因となって結果を左右しているのかを分析しました。

02. 製品アイデアの考案

分析の結果を参考にしつつ、製品のアイデアを考案しました。

03. ウェブデザイン

Figmaを用いてウェブデザインを行いました。

分析結果を参考にしつつ、ウェブサイトに載せる情報や並べる順番に配慮し、製品テーマと合うような色・フォントでデザインを行いました。

04. コーディング

ウェブサイト制作の担当が2人だったため、GitHubを用いてコード管理を行いながらコーディングを行いました。

Product

Features

01, 製品名・コンセプトをトップに

サイトを訪れた人が一番初めに目にする場所に製品名とコンセプトを配置することによってサイトを訪れた人に製品に対する興味を持ってもらいやすくするように工夫。

02, コンテンツの並べ方

製品紹介→目標→製品について→製品の使い方→リターン紹介という流れでWebを作成した。この順番にすることによってサイトを訪れた人は製品について少しずつ知りながら、最終的に投資するかを決めることができるようにしました。

03, 色とフォント

私たちの製品は医療品というカテゴリーに入ると考えたためWebサイトは清潔感のある雰囲気を目指しました。青や緑を基調として作成し、フォントも読みやすく、サイトの雰囲気に合ったものを選んで使用しています。

04, 快適な閲覧を

レスポンシブに対応させ、画像を"webp"形式にして軽量化し、スクロールに合わせて画像などを読み込む「遅延読み込み」を行うことでサイトの読み込み時間を短縮するなどを工夫で快適に閲覧できるようにしています。
