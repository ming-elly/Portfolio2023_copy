プロジェクト学習「グローバルデザイン」紹介用ウェブサイト


制作人数：2名　　担当：ウェブデザイン/コーディング

参加時期：2022年11月～2022年12月　　　使用アプリ・技術：Figma / VSCode / HTML,CSS / jQuery / GitHub / loose drawing

Process

01. 必要情報の収集・整理

ウェブサイトに掲載する情報の収集と整理を行いました。

02. Figmaでのウェブデザイン

Figmaを用いてウェブデザインを行いました。

PC版サイトのデザインを行ったのちに、スマートフォン版サイトのデザインを行いました。

03. コーディング

ウェブデザインをもとにコーディングを行いました。

ウェブサイト内の一部にアニメーションをつけるため、jQueryなども使用しました。

Product

Features

01, 初めて見る人に

このウェブサイトはプロジェクト学習での1年間の活動を広報を目的としたウェブサイトであり、私たちの活動内容を全く知らない人が訪れます。そのため、私たちの活動を容易に理解するために情報を配置する順番に気を付けました。

02, 動きを取り入れる

訪れる人に楽しんでもらうために、ウェブサイト内にアニメーションを用いることで動きのあるウェブサイトにしました。グローバルという私たちの活動の軸を表現するために雲や飛行機のイラストを使用しています。

03, 空白の取り方

訪れた人が見やすくなるように広めに空白を取っています。特にアプリケーションを紹介する”Cuttie”のページでは、特徴と使用方法を紹介する部分で1画面に1つの情報が載るように気を付けました。

04, レスポンシブ対応

1年間の活動を学内の学生や教員だけでなく、一般の方にも報告する最終発表会が行われました。その際に私たちの発表に訪れた人にスマートフォンなどのモバイル端末からウェブサイトを気軽に閲覧してもらえるようにレスポンシブ対応をしました。
