What is

「コロコロ太陽系」ウェブサイト

インタラクティブなピタゴラ装置 / 紹介用ウェブサイトの制作


制作：グループワーク（4人）　　　　　　担当：ウェブデザイン / コーディング

制作時期：2021年11月～2021年12月　　　使用アプリ・技術：Adobe XD / Adobe Premiere Pro / HTML, CSS / VSCode / GitHub

Process

01. ピタゴラ装置のテーマ考案

Miroにて、ブレインストーミングの形式でピタゴラ装置のテーマ案を考案しました。対象者や設置する場所を考えつつ、テーマ案を出していきました。

02. 装置の製作

決まったテーマ案をもとにして、装置のギミックや形を考えた上で製作を進めていきました。

03. ウェブデザイン

上記の工程と並行して、Adobe XDを用いてウェブデザインを行いました。

初のウェブサイト制作だったため、様々なサイトを参考にしながらデザインを行いました。

04. コーディング

ウェブサイトのコーディングは初だったため、オンライン学習サービスのProgateや書籍などを使って知識を付けながらコーディングを行いました。

Product

Features

01, トップページにはコンセプトを

サイトを訪れた人が初めに見るトップページに作品名と作品コンセプトを載せることによって、訪れた人が作品に興味を持ってもらえるようにしています。

02, 背景に動画を使用

ヒーローヘッダーの背景に画像ではなく、動画を使用しました。このようにすることによって画像よりも多くの情報を伝えることが可能で、サイトを訪れた人により興味を持ってもらえると考えました。

03, 作品紹介ページのレイアウト

作品のギミックを紹介する際に、これら全ての特徴をただ羅列するだけではウェブサイトが見にくくなってしまうと考えたためFlexコンテナーを用いて特徴紹介の部分をより見やすいようにしました。

04, Galleryページ

作品にある多くの特徴や外観をより良く伝えることを可能にしたいなと考えました。そのため、敢えてこのページには文章を書かないで、作品の写真・ロゴ・説明の動画のみを配置したページを制作しました。
