今までの作品をまとめたポートフォリオウェブサイト


制作：個人　　担当：ウェブデザイン / コーディング

制作時期：2022年12月～　　　使用アプリ・技術：Figma / STUDIO / VSCode / HTML, CSS / jQuery / GitHub

Product

Web Portfolio, 01 / Features

01, ウェブ作品をより伝える

ウェブサイトは写真などにするととても細長くなってしまったりする影響で、紙用ポートフォリオとの相性が良くないと感じていました。そのため、ウェブデザインやコーディングの作品についてはウェブポートフォリオに載せることでどのような作品かをより伝えられるようにしました。

02, シンプルに見やすく

初期のデザイン案ではポートフォリオのトップページにそれぞれの作品のウェブサイトを全て写真として載せていましたが、最終的にはモックアップ画像を使用して、よりシンプルにしました。ウェブサイトの写真はそれぞれの個別ページに移動しました。

03, アニメーションで分かりやすく

ナビゲーションの部分や、ポートフォリオのトップページから各作品のページに飛ぶ際にリンクを踏んでいることが分かるように色の変化やサイズの変更などのアニメーションを付けました。このようにすることでサイトを閲覧する人にとってリンク部分を分かりやすくなるようにしました。

04, STUDIOでの作成

このウェブサイトはノーコードでウェブサイトを作ることができる"STUDIO"を使用して作成しています。理由としては、コーディングによる縛りがない為、自分の思ったようにウェブサイトを作成できると考えた点や近年需要が高まってきているノーコードでの制作ツールを使用するのも経験として重要だと考えた為です。

Web Portfolio, 02 / Features

01, 快適な閲覧

ポートフォリオサイトという特性上、写真が多くなってしまいますが写真を".webp形式"にし、"loading=lazy"を用いることで遅延読み込みを入れてスムーズに閲覧ができるように工夫しています。

02, たくさんの情報をコンパクトに

jQueryプライグインのslick.jsなどを用いたスライドを使って、出来るだけコンパクトに成果物の写真などをまとめています。

03, リンク部分などのアニメーション

ヘッダーナビゲーションや各作品へ飛ぶためのリンクなどをホバーした際にアニメーションを付けることで、視覚的に分かりやすくなるように配慮しました。
