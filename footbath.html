What is ...

足湯×ラーメンの新提案「麺処　ふっとばす」

温泉をテーマにしたグループワークにて、あまり温泉に行かない男子大学生をターゲットとして考案しました。


制作：グループワーク（4人）　　　　　　担当：プロダクト案 / 建物内装のデザイン / 模型製作

制作時期：2022年11月～2023年1月　　　使用アプリ・技術：Figma

Process

01. フィールドワーク

函館市にある温泉街「湯の川」で実際に温泉施設を訪れてフィールドワークを行いました。

温泉内のレイアウトや、温泉を上がった後の休憩場所など様々な場所の観察をしました。また、それだけではなく街の雰囲気などについても観察を行いました。

02. 案出し

具体的な案を出す前にターゲットを「あまり温泉に行かない男子大学生」としました。

フィールドワークを行って感じたことや、分かったことなどを挙げながら、どのようなモノならばターゲットに合ったモノになるかを考えました。また、案を出すのと並行してGoogleFormを使ったアンケート調査なども行い、男子大学生の温泉への関り度合いなども考えました。

03-1. 成果物の制作(Minecraft)

話し合いの結果、私たちのグループで成果物として見せるものは、店の模型/3Dモデリングなどになりました。しかし、模型や3Dモデリングを作る前にメンバー間でプロダクト案に対するイメージをすり合わせる必要がありました。

そこで私たちはMinecraftというサンドボックス型のゲームを用いてプロトタイピングを行うことにしました。この方法を選んだ理由として、モデリングソフトとは違って、共同作業が可能であり、難しい前提知識を必要としないため、建物のプロトタイピングをするにあたって適していると考えたからです。

03-2. 成果物の制作(模型)

模型の制作には、MDF素材を主に用いて制作しました。

レーザーカッターを用いて、MDFを加工し、模型の部品にしたのちに油性インクを用いて色を付けました。このようにして出来上がった部品を、木工用ボンドやグルーガンなどを使って組み立てました。

04. 成果物発表/展示会

本講義では、最終週に成果物発表会を兼ねた展示会が行われました。

私たちのグループでは、制作した模型やラーメン店を模した足湯体験のコーナーを作って展示会を行いました。

Product

Minecraft Prototype / Features

01, 和風な建築

温泉の和風な雰囲気を出すために建物は木材などをメインとしています。

02, 温泉に入るプロセスを再現

銭湯などの施設と同じく、入口で靴を脱ぐ動作を取り入れることで温泉に入るプロセスを再現しています。

03, 屋外席を用意

寒暖差を感じながら足湯と食事を楽しめるように屋外席を用意しています。

04, 様々なタイプの席を用意

カウンター席・座敷席・テーブル席・屋外席と様々なタイプの席を用意しています。1人で来る大学生や大人数で来る大学生など様々なタイプのお客さんを想定しています。

Model Prototype / Features

01, 素材

模型のメイン素材として、MDFを使用しています。レーザーカッターでの加工が可能な点や、色塗りが容易な点を考慮して選びました。

02, 水の質感の再現

足湯部分の水の質感を再現するためにレジン液を用いています。

03, 部品ごとに組み合わせ

室内 / 屋外席 / キッチン部分がそれぞれ部品として独立しており、これらを組み合わせることで模型が完成するようにしています。
